# ML Optimize Pro

Suite premium de otimização WordPress da **ML Lopes Design**.

Cache de página, minify CSS/JS, defer/delay JavaScript, remove unused CSS, lazy load, lazy render, Script Manager (estilo Perfmatters), Bloat Remover, self-host Google Fonts, preload de recursos, Speculation Rules, Database cleanup, Heartbeat control, CDN rewrite, Performance Hub com score CWV estimado e auto-update via GitHub.

## Instalação

Baixe o ZIP da [última release](https://github.com/mlopesdesign/ml-optmize-pro/releases/latest) e faça upload em `Plugins > Adicionar novo > Upload`.

O plugin é premium e distribuído fora do WordPress.org. Auto-update via GitHub funciona nativamente — o painel WP mostra a notificação de update quando uma nova tag `v*` é publicada aqui.
